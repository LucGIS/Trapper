Welcome to Trapper!

Trapper is a web-based content management system aimed at processing and analysis of the media resources on wildlife and animal behavioral patterns. The project is an intiative of the Mammal Research Institute of the Polish Academy of Sciences, with the collaboration of Bialystok University of Technology.

Please login to see more features of the system. If you have suggestions on how to improve Trapper you can visit us on our IRC channel: #trapper on irc.freenode.net.

In case of problems, you can also submit a message to the user admin using the built-in messaging system.

We hope you will have fun using the system.
-Trapper development team.
